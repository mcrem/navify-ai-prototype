# Navify Analytics AI Companion Prototype

Stakeholder-ready front-end concept prototype showing how an AI Companion could be embedded into a future Navify Analytics experience.

## Run locally

```bash
npm --prefix web install
npm --prefix web run dev
```

Then open `http://localhost:4173/`.

## What is included

- Mocked analytics dashboard inspired by Navify Analytics
- Embedded AI Assistant entry point inside the core experience
- Animated transition into a premium AI interaction surface
- Apple Intelligence-inspired gradients, glow, translucency, and motion
- Responsive layout for desktop and mobile concept review

## Notes

- No backend is used
- All analytics and AI responses are mocked for concept validation
